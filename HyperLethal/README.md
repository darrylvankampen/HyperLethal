# HyperLethal

HyperLethal is an SPT mod that adds overpowered items. At this moment mostly focussed on ammo, but other items might come in the future. Who knows?!

## Features

- Uses [WTT-CommonLib](https://github.com/WelcomeToTarkov/WTT-CommonLib/) to create new custom items
- Config can be done in /db/

## Requirements
- SPT 4.0.0+
- NET SDK 9.0
