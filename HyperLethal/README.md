# HyperLethal

HyperLethal is an SPT mod that adds overpowered items. At this moment mostly focussed on ammo, but other items might come in the future. Who knows?!

## Features

- Registers "overpowered" ammo by cloning exisiting templates
- Applies custom stats (damage, pen, etc..)
- Can inject these items into traders (currently Peacekeeper)
- Includes a bare bone validation system

## Requirements
- SPT 4.0.0+
- NET SDK 9.0
